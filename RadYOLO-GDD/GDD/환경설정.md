# 환경설정

## 개발 환경
- IDE: VS Code
- Python 환경: Anaconda
- GPU: RTX 4060 Laptop (개발), 연구실 데스크톱 (메인 연산)
- CUDA Toolkit 필요

## 툴체인
| 도구 | 용도 |
|---|---|
| VS Code | 전체 코딩 |
| mmWave Studio | 레이더 초기 설정/검증 |
| MATLAB | 신호처리 알고리즘 개발/검증 단계 |
| DCA1000 GUI | Ethernet 연결 설정 |
| Git | 버전 관리 |

## 캘리브레이션
- 카메라-레이더 외부 캘리브레이션 (1회)
- 체커보드 + 코너 리플렉터 사용
- 변환 행렬 저장 후 재사용

## 기술 스택
| 역할 | 기술 |
|---|---|
| 객체 감지 | YOLOv8-seg |
| 깊이 추정 | Depth Anything v2 |
| 배경 복원 | SD Inpainting |
| 사람 3D | HMR2.0 + SMPL |
| 사물 3D | Tripo3D / Rodin API |
| 사물 포즈 | FoundationPose |
| 레이더 처리 | Python + NumPy |
| 렌더링 | Open3D |

## 디렉토리 구조

```
RadYOLO/
  scripts/
    calibrate.py            # 카메라-레이더 변환행렬 계산 (1회)
    build_background.py     # background/pipeline.py 호출하는 CLI 래퍼

  src/
    main.py                 # 실시간 루프 진입점

    radar/
      capture.py            # DCA1000 Ethernet 수신
      signal.py             # Range/Doppler FFT, CFAR, Angle 추정

    camera/
      capture.py            # 카메라 프레임 수신
      yolo.py               # YOLOv8-seg (bbox + seg mask)
      depth.py              # Depth Anything v2

    background/
      pipeline.py           # 배경 생성 오케스트레이션 (초기화/재실행 공용)
      scale.py              # 선형회귀 (a·b 계산/로드)
      points.py             # 배경 포인트 저장/교체

    fusion/
      match.py              # 레이더↔YOLO bbox+mask 매칭
      unknown.py            # 레이더 미지 물체 감지

    modeling/
      human.py              # HMR2.0 + SMPL
      object.py             # API 메시 요청 + FoundationPose

    rendering/
      viewer.py             # Open3D 뷰어 (좌우 분할)
```

### 배경 갱신 트리거
- YOLO 재학습 완료 → `background/pipeline.py` 전체 재실행
- 인페인팅 영역 노출 → `background/points.py` 부분 교체만
