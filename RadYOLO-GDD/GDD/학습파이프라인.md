# 학습파이프라인

## 목적
레이더가 감지했지만 YOLO가 모르는 물체를 자동으로 학습하여 YOLO 업데이트

## 흐름
1. 레이더 Doppler → 움직이는 물체 감지
2. 해당 위치에 YOLO 감지 없음 → "unknown" 물체로 분류
3. 크롭 이미지 수집 + 자동 라벨링
4. 트리거 조건 충족 시 → YOLO 파인튜닝 (백그라운드)
5. 업데이트된 모델 배포

## 트리거 조건 (잠정)
- 유효 프레임 100장 이상
- 각도 변화 20도 이상 커버
- FoundationPose 신뢰도 0.7 이상 프레임만 사용
- (실제 수치는 실험 후 조정)
