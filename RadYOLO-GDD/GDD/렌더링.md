# 렌더링

## 렌더러
- Open3D (프로토타입 ~ 메인)
- Unity + Python 소켓 브릿지 (고품질 단계, 선택)

```
┌─────────────────────┬──────────────────────────┐
│   카메라 뷰         │   Open3D 3D맵             │
```

## 배경 처리
- 초기화 시 SD Inpainting으로 객체 제거 → 배경 포인트클라우드 생성
- 객체 사라지면 실제 카메라 이미지로 해당 영역 교체
