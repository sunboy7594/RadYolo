# RadYOLO

## 개요
카메라(YOLO + Depth Anything) + mmWave 레이더(IWR6843ISK)를 융합하여 실시간 3D 맵을 렌더링하는 시스템

## 프로젝트 종류
기술 연구 / 실험 프로젝트

## 카테고리
- 하드웨어 — 레이더 보드, 카메라, DCA1000 스펙/설정
- 파이프라인 — 전체 데이터 흐름, 융합 구조
- 신호처리 — 레이더 FFT/CFAR/Angle 추정
- 객체감지 — YOLO, Depth Anything, FoundationPose
- 3D모델링 — SMPL/HMR2.0, API 메시 생성
- 렌더링 — Open3D 3D맵 구성
- 학습파이프라인 — YOLO 자동 파인튜닝 구조
- 환경설정 — 개발 환경, 툴체인, 캘리브레이션
