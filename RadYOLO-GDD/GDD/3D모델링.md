# 3D모델링

## 사람
- HMR2.0 → SMPL 체형 + 포즈 파라미터 실시간 추출
- 카메라 이미지 → 텍스처 투영
- 결과: 실제 체형에 가까운 리깅된 3D 인체, 포즈 실시간 구동

## 사물
- 최초 감지 시: 크롭 이미지 → Tripo3D / Rodin API → 3D 메시 (5~10초)
- 대기 중: 포인트클라우드로 임시 표시
- 메시 수신 후: FoundationPose로 6DoF 매 프레임 업데이트
- 관절 있는 사물: 미구현 (추후 검토)

## API 후보
| API | 속도 | 비고 |
|---|---|---|
| Tripo3D | ~5~10초 | Python SDK 있음 |
| Rodin (Hyper3D) | ~10초 | 디테일 강함 |
| Meshy | ~10~30초 | 텍스처 품질 좋음 |
