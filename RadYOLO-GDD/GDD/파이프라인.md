# 파이프라인

## 전체 구조

### 초기화 (1회)

**1. 카메라-레이더 캘리브레이션**
- 변환 행렬 저장

**2. 배경 포인트클라우드 생성**

2-1. 다중 프레임 수집
- 카메라로 N프레임 촬영 (프레임마다 노이즈/조명 미세 차이 활용)

2-2. 객체 제거
- YOLOv8-seg → 객체 픽셀 마스크
- SD Inpainting → 배경 이미지 N장

2-3. 상대 깊이맵 생성
- DA V2 (상대 버전) → 깊이맵 N장
- Metric 미사용 이유: 환경 무관 일반화 우선

2-4. 레이더 배경 포인트 수집
- IWR6843ISK, static clutter 필터 OFF
- 캘리브레이션 변환행렬로 레이더 포인트 → 카메라 좌표계 변환
- YOLO 마스크 영역 포인트 제외 → 배경 포인트만 추출

2-5. 검증 + Scale 보정 (동시)
- N프레임 각각: (DA 상대값, 레이더 절대값) 쌍 → 선형 회귀
- `depth_abs = a * depth_rel + b`
- R² 계산 → R² 최고 프레임 선택, a·b 확정

2-6. 컬러 포인트클라우드 생성
- `depth_abs = a * depth_rel + b`
- back-projection: `X = (u-cx)*Z/fx`, `Y = (v-cy)*Z/fy`
- 색상: 원본 카메라 이미지 RGB
- 결과: (X, Y, Z, R, G, B) 저장 / 픽셀당 1점 / 사물 뒤 공간은 빈 공간

### 실시간 루프
- 레이더: DCA1000 raw ADC → 신호처리 → 포인트클라우드
- 카메라: YOLOv8-seg + Depth Anything → 깊이맵 보정
- 융합: 레이더 포인트 ↔ YOLO bbox 매칭
  - 사람 → HMR2.0 + SMPL
  - 사물 → API 메시 + FoundationPose 6DoF
  - 미지 물체 → 누적 후 YOLO 파인튜닝 트리거

### 배경 갱신

| 트리거 | 조건 | 처리 |
|---|---|---|
| YOLO 재학습 완료 | 즉시 | 초기화 2번 전체 재실행 |
| 인페인팅 영역 노출 | 넓이 ≥ threshold AND N프레임 지속 | DA V2 전체 실행 → 해당 영역 포인트만 교체 (기존 a·b 재사용) |

- threshold, N프레임: 실험으로 튜닝
- 인페인팅 영역 노출 시 실제 카메라 이미지 사용 (인페인팅 불필요)

## 역할 분담
| 센서 | 역할 |
|---|---|
| 레이더 | 절대 거리, 속도, 미지 물체 감지 |
| 카메라 | 객체 감지/분류, 상대 깊이 |
| 융합 | 절대 거리 기반 3D맵 |
