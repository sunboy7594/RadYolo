# 신호처리

## 수신 방식
- DCA1000 raw ADC → Ethernet (192.168.33.180:4098) → Python NumPy 처리
- MATLAB은 알고리즘 검증 단계에서만 활용 (교수님 코드 참고)

## 처리 파이프라인
1. Range FFT — chirp 방향 FFT → 거리 추출
2. Doppler FFT — chirp간 위상차 FFT → 속도 추출
3. CFAR 검출 — 노이즈 대비 피크 선별 → 실제 물체 포인트
4. Angle 추정 — RX 안테나간 위상차 → azimuth + elevation

## 파라미터 (실측값)
| 항목 | 값 |
|---|---|
| ADC samples | 256 |
| chirp loops | 128 |
| RX | 4 |
| frames | 8 |
| BW | 1798.92 MHz |
| Fc | 77 GHz |
| Sample Rate | 10000 ksps |
| Ramp End Time | 60 μs |

## 출력
- (range, azimuth, elevation, velocity, intensity) 포인트클라우드
