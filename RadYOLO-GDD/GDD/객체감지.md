# 객체감지

## YOLOv8-seg
- 객체 감지 + 픽셀 단위 마스크
- 사람/사물 분류

## Depth Anything v2
- 단안 카메라 상대 깊이맵 추출
- 레이더 scale factor로 절대 깊이 보정

## FoundationPose
- 입력: RGBD + 3D 메시
- 출력: 6DoF 포즈 (위치 + 회전) 매 프레임
- 렌더-앤-비교 방식으로 임의 신규 객체 즉시 적용

## 레이더 미지 물체 감지
- Doppler 필터로 움직이는 물체 자동 감지
- YOLO 미감지 물체 → 크롭 → API 3D 생성 → 누적 학습
