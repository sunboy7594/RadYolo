## 캡처 보드

## 카메라
- 웹캠 (프로토타입 기준)

## 연결 구성
```
AWR2243BOOST USB → PC (COM5: Application UART, COM3: Auxiliary Data Port)
AWR2243BOOST LVDS ↔ DCA1000EVM
DCA1000EVM Ethernet → Dell WD15 도킹스테이션 → PC
DCA1000EVM 전원: 5V 외부 어댑터
```

## mmWave Studio 설정 순서
1. Device Variant: XWR22xx / Operating Frequency: 77GHz → Set
2. COM5, Baud Rate 115200 → Connect → Device Status: XWR2243/ASIL-B/SOP:2/ES:1.1 확인
3. BSS FW: `radarss\xwr22xx_radarss.bin` → Load
4. MSS FW: `masterss\xwr22xx_masterss.bin` → Load
5. SPI Connect
6. RF Power-up
7. SetUp DCA1000 → Connect, Reset and Configure → FPGA Version: 2.8 확인
8. StaticConfig 탭 → RF Init → `RF Init async event received! Status: Passed` 확인
